## Description
This project analyzes the relationship between historical redlining (HOLC grades) and modern vegetation health (NDVI) in Fresno, CA. It uses spatial data science tools including ibis, leafmap, rasterio, and exactextract.

## Key Features
- Spatial map with two layers (NDVI COG + HOLC polygons)
- Full reproducibility with COG creation and URL-based data access
- Statistical summaries and visualizations
- Scientific write-up included in the notebook
- GitHub Pages web map deployment
